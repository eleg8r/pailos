# PAILOS — Personal AI Learning Operating System

## 1. Core Concept

System that provides:

* adaptive tutoring
* knowledge graph
* memory tracking
* goal-driven learning
* real-world integration (code, logs)
* multi-agent personalities

---

## 2. Architecture

* CLI (Node)
* Web UI (React)
* Tutor Engine
* Memory System
* Knowledge Graph
* Analytics Engine
* MCP Skills Layer

---

## 3. Core Features

### Tutor CLI

* start session (math, coding, debate)
* adaptive loop (diagnose → teach → review)

### Memory System

* tracks skill levels
* tracks weak areas

### Knowledge Graph

* concepts + dependencies
* guides learning order

### Adaptive Engine

* selects next topic
* adjusts difficulty

### Multi-Agent Personalities

* strict
* playful
* challenger

### Analytics

* skill heatmap
* retention tracking

### Autonomous Tutor

* triggers sessions
* sends recommendations

### MCP Skills

* repo search
* SQL inspection
* log analytics

---

## 4. Example Flow

1. User starts session
2. System loads memory
3. Graph determines next concept
4. Persona selected dynamically
5. Claude teaches interactively
6. System updates memory + analytics

---

## 5. Tech Stack
